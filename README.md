# Curated speeches by President Obama

This app will allow users to read curated speeches delivered by President Obama during his two terms in office.

## Technology used

* [WordPress REST API](http://v2.wp-api.org/)
* [React](https://facebook.github.io/react/)
* [Redux](http://redux.js.org/)
* [React Router](https://github.com/ReactTraining/react-router)
* [Thunk](https://github.com/gaearon/redux-thunk)
